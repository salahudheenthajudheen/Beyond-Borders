# Startup Readiness Score

> Measuring how ready an opportunity is to become a fundable, buildable startup.

---

## Overview

The Startup Readiness Score (SRS) is a structured assessment tool used in Beyond Borders to evaluate how ready a problem statement or startup concept is for actual execution.

Unlike the Evaluation Framework (which scores the opportunity itself), the SRS scores the *readiness of a team to pursue the opportunity* — covering problem understanding, solution clarity, team capability, market knowledge, and execution plan.

---

## The Six Dimensions

### 1. Problem Clarity (0–10)
How well does the team understand the problem?

| Score | Description |
|---|---|
| 9–10 | Spent 50+ hours with problem owners; have primary research data |
| 7–8 | Interviewed 10+ people who face the problem; documented findings |
| 5–6 | Basic research done; some direct exposure to the problem |
| 3–4 | Problem understood mostly from secondary research |
| 1–2 | Assumed problem based on general knowledge |

### 2. Solution Clarity (0–10)
How specific and validated is the proposed solution?

| Score | Description |
|---|---|
| 9–10 | Working prototype tested with real users |
| 7–8 | Detailed solution design with user validation of key assumptions |
| 5–6 | Solution concept clear; key design decisions not yet validated |
| 3–4 | High-level solution concept only |
| 1–2 | No clear solution design |

### 3. Technical Capability (0–10)
Does the team have the skills to build the solution?

| Score | Description |
|---|---|
| 9–10 | Full-stack capability in the team; has built similar systems before |
| 7–8 | Core capability present; some skill gaps with a hiring plan |
| 5–6 | Adjacent capability; significant learning required |
| 3–4 | Major skill gaps with no clear resolution |
| 1–2 | Team cannot build this without near-complete outsourcing |

### 4. Market Knowledge (0–10)
How well does the team understand the market?

| Score | Description |
|---|---|
| 9–10 | Knows the buyers, buying process, competitive landscape, and pricing intimately |
| 7–8 | Has mapped key customers and has LOIs or pilots underway |
| 5–6 | Has identified target customers; sales process not yet understood |
| 3–4 | Market identified; no direct customer conversations |
| 1–2 | Market assumptions based on desk research only |

### 5. Business Model (0–10)
How clear and validated is the revenue model?

| Score | Description |
|---|---|
| 9–10 | First paying customers; unit economics validated |
| 7–8 | Pricing validated with customer conversations; model is clear |
| 5–6 | Revenue model chosen; pricing not validated |
| 3–4 | Multiple revenue model options; no clear decision |
| 1–2 | No clear revenue model |

### 6. Execution Readiness (0–10)
Is the team ready to move fast?

| Score | Description |
|---|---|
| 9–10 | Full-time committed team; funded; has a 90-day plan |
| 7–8 | Part-time but serious; clear path to full-time; 90-day plan exists |
| 5–6 | Team interested but not committed; plan is vague |
| 3–4 | Individual contributor; no team formed |
| 1–2 | Idea stage; no execution commitment |

---

## Score Interpretation

| SRS Score | Stage | Recommended Next Step |
|---|---|---|
| 8.0–10 | Startup Ready | Apply to accelerator; begin fundraising |
| 6.5–7.9 | Pre-Startup | Address weakest dimension; 90-day sprint to next level |
| 5.0–6.4 | Concept Validation | Run structured validation experiments |
| 3.5–4.9 | Idea Stage | Deep customer discovery before building anything |
| Below 3.5 | Exploration | Read, research, talk to domain experts |

---

## Using the SRS in Beyond Borders

When submitting a startup opportunity to this repository:

1. Self-assess your SRS score across all 6 dimensions
2. Include the score and evidence in your submission
3. Reviewers will calibrate during the PR review process
4. Use the score to identify what work is needed before your next milestone
