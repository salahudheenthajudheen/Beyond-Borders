# Netrasemi — Open Projects

This folder contains open-source project ideas and active projects built on Netrasemi's Edge AI technology.

Projects range from reference implementations and open datasets to full prototype systems and developer tools.

---

## How to Add a Project

1. Copy the [Project Template](../../../project-template.md)
2. Name your file: `[vertical]-[short-description].md` (e.g., `agriculture-crop-disease-detector.md`)
3. Fill in all sections completely
4. Open a pull request
